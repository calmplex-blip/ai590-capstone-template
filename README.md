# AI590 Masters Project in Artificial Intelligence

This individual Classroom 50 repository carries one capstone through four cumulative milestones introduced in Weeks 1, 3, 5, and 7. Canvas is the official submission, rubric, feedback, and grade record. Week 7.5 adds no quiz, Reading, or new Classroom 50 assignment; it is reserved for final demonstration, submission, and feedback.

## Evidence standard

Connect every material claim to authorized data, reproducible code or analysis, captured output, evaluation, a responsible-use boundary, and a decision record. The final project must run from a clean environment and disclose limitations, monitoring, recourse, and the condition that would require the conclusion to change.

## Required paths

- problem_statement.md, literature_review.md, and project_charter.md
- data_card.md and architecture_decision.md
- notebooks/ for baseline, pipeline, model, and final-experiment evidence
- evaluation_report.md and responsible_ai_record.md
- final_report.md, presentation/final_defense.md, and handoff_runbook.md

If only one student is enrolled, instructor or simulated-stakeholder review replaces peer interaction. Preserve each perspective, question, response, revision, and test.
